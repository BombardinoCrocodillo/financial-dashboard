# Дашборд: Процесс управления инвестициями

Интерактивный инструмент для анализа инвестиционных портфелей.

## Возможности
- Загрузка CSV с данными об инвестициях
- Фильтрация по портфелю и уровню риска
- Визуализация:
  - Доходность по типам активов
  - Распределение портфеля
  - Сравнение вложений и текущей стоимости

## Структура данных (CSV)
Колонки: `date`, `portfolio`, `investment_type`, `initial_investment`, `current_value`, `return_percent`, `risk_level`

## Запуск
```bash
pip install -r requirements.txt
python app.py
